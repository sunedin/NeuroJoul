Core Stack

NeuralJoule runs an event-driven architecture where edge agents execute deterministic safety loops, cloud agents coordinate learning and policy rollout, and market adapters perform autonomous bidding.

Edge Layer

  • Battery Management Integration (BMS + inverter telemetry).
  • Local inference runtime for A3 Asset Agents.
  • Sub-millisecond protection loops and fault rerouting.

Coordination Layer

  • Orchestrator policy engine for safety and economics balancing.
  • Multi-agent state synchronization and command arbitration.
  • Digital twin simulation for scenario rehearsal.

Market Layer

  • Utility API connectors for reserve and frequency products.
  • Swarms executing MARL-based bidding strategies.
  • Settlement and audit trails for machine-to-machine transactions.

Sustainability Layer

  • Carbon intensity streams fused with dispatch logic.
  • REC generation, validation, and trading automation.
  • Impact scorecard tied to every dispatched MWh.

Agent Topology

The Orchestrator supervises three specialized agent classes. Asset Agents prioritize physical health, Trading Swarms maximize revenue, and Carbon Proxy Agents enforce emissions-aware decisions.

Execution Flow

  • 1) Edge agents ingest telemetry and score equipment state.
  • 2) Orchestrator applies policy constraints and confidence thresholds.
  • 3) Swarm submits market actions and receives settlement outcomes.
  • 4) Carbon agents validate impact and record REC eligibility.

Reliability Model

  • Fail-safe local controls continue if cloud links degrade.
  • Command queues are idempotent and replay-protected.
  • Policy updates are versioned and can be rolled back instantly.
